# Class-Scheduling

A Java-based **Class Scheduling** application developed in **Eclipse** that helps organize and manage academic schedules. The project focuses on building structured schedules while reducing common issues such as overlapping class times, limited room availability, and instructor/student conflicts.

## Overview
This program models real scheduling components (courses, time slots, rooms, instructors, and/or students) using **object-oriented design**. It allows users to create, view, and adjust schedules in a clear way, making the scheduling process more efficient and reliable.

## Features
- Create and manage class schedule entries (courses, times, rooms, etc.)
- Detect and prevent scheduling conflicts (ex: overlapping times)
- Organized data structures using Java’s Collections Framework
- Input validation and error handling for more stable program behavior
- Designed with scalability in mind for larger schedules

## Built With
- **Java**
- **Eclipse IDE**
- Object-Oriented Programming (OOP)

## How to Run (Eclipse)
1. Clone or download this repository.
2. Open **Eclipse** → `File` → `Open Projects from File System...`
3. Select the project folder and import it.
4. Locate the `main` class (the file containing `public static void main(String[] args)`).
5. Right-click the file → `Run As` → `Java Application`
